关于nodejs的一二事

2017-6-2

Node.js 是一个基于 Chrome V8 引擎的 JavaScript 运行环境。
Node.js 使用了一个事件驱动、非阻塞式 I/O 的模型，使其轻量又高效。
Node.js 的包管理器 npm，是全球最大的开源库生态系统。

nodejs

1.nodejs的安装

在http://nodejs.cn/网站下载个人电脑适应的nodejs版本后，双击进行安装， 安装过程中可自行选择安装路径和安装模式。

❀ 在win8.1操作系统中，.msi版本安装可能会出现2503或者2503之类的错误，此时只需将安装文件放在你所知的目录下， 利用管理员权限打开命令提示符界面，msiexec /i 文件目录:\文件名.msi，例如：msiexec /i c:\1.msi

安装完成后，在命令提示符界面输入node -v查询node版本，输入npm -v查询npm版本。
